เกี่ยวกับราคา GPU

MSRP หมายความว่าอย่างไร

ด้วยการประเมินที่เพิ่มขึ้นของสกุลเงินดิจิตอลเข้ารหัส ราคา GPU ได้ปีนขึ้นไป 2-4x ของพวกเขา MSRP . บางคนกล่าวหาว่าโควิดและการล็อกดาวน์ที่บังคับให้คนที่บ้านมีความต้องการเครื่องใช้ในบ้านเพิ่มขึ้น อย่างไรก็ตาม เกือบ 1:1 correlation between historical prices of gpus and cryptos makes covid almost irrelevant.

แล้วแมชชีนเลิร์นนิงล่ะ?

ในการเปรียบเทียบ แมชชีนเลิร์นนิง (หรือโดยเฉพาะอย่างยิ่ง การเรียนรู้เชิงลึก) จะทำให้ช่วงราคาระดับไฮเอนด์อิ่มตัวเท่านั้นเพราะจำเป็น FP ความแม่นยำซึ่งเป็นข้อกำหนดเฉพาะมากขึ้นโดยที่ crypto พลัง GPU พยายามที่จะใช้ประโยชน์จาก ทั้งหมด ชุดคำสั่ง GPU ดังนั้นกลุ่มผลิตภัณฑ์ทั้งหมดของตลาด GPU จึงน่าสนใจสำหรับการขุด

เราสามารถคาดหวังให้ราคา GPU กลับไปสู่ ​​MSRP ได้หรือไม่?

ไม่ ตลาด GPU ถูกกินโดยการขุด crypto และจะไม่มีวันฟื้นตัว

  • GPU คือ วางซ้อนกันได้ซึ่งหมายความว่าคนขุดแร่โดยเฉลี่ยของคุณซื้อ GPU อย่างน้อย 4 ตัว โดยที่ผู้บริโภคทั่วไปของคุณต้องการเพียง 1 GPU ส่วนใหญ่

  • หาก cryptocurrencies หลักเปลี่ยนเป็น POS (ชอบ ผลประโยชน์ทับซ้อน ) สกุลเงินดิจิตอลเข้ารหัสอื่น ๆ จะใช้อำนาจการแฮช การแก้ไขราคาที่คมชัดยิ่งขึ้นจะต้องเกิดขึ้นเพื่อทำให้ GPU ขายโดยนักขุดอย่างกะทันหัน

  • คนงานเหมืองกำลังเปิดและปิดอุปกรณ์การทำเหมืองที่ดี ขึ้นอยู่กับอัตราส่วนผลตอบแทนระหว่างค่าไฟฟ้าและราคาสกุลเงินดิจิตอล พวกเขาจะไม่ขายฮาร์ดแวร์ในครั้งแรก

      ทำอะไร?

      เฉพาะ GPU ที่มีราคาสมเหตุสมผล ณ เวลาที่เขียนเท่านั้นและแล็ปท็อปที่มาพร้อมกับ GPU (ใน มือถือ รุ่น) หากผู้ผลิตต้องการเปิดตลาด GPU อีกครั้งสำหรับผู้ใช้ทั่วไป พวกเขาต้องนำเสนอโซลูชั่นแบบบูรณาการที่ แข่งขันได้มากขึ้น กับผู้อุทิศตน หรือออกแบบ GPU ชนิดอื่นที่อยู่กึ่งกลางระหว่างเฉพาะและแบบบูรณาการ อย่างไร? โดยพิจารณาถึง CPU และ GPU ที่เชื่อมต่อกันอย่างแน่นหนาซึ่งยังเป็นแบบแยกส่วนได้ (และสามารถเปลี่ยนได้ แต่วางซ้อนกันไม่ได้!)
