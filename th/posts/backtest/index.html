กรอบงาน Backtesting ของ Julia

ซื้อขาย cryptocurrencies กับจูเลีย

Backtest.jl เป็นโครงการจูเลียที่ฉันเขียนเพื่อทำการซื้อขายสกุลเงินดิจิทัล ขณะนี้มันขอกับ ccxt ไลบรารี python เพื่อเชื่อมต่อกับการแลกเปลี่ยน apis สำหรับตอนนี้เพียงเพื่อดึงข้อมูล ohlcv

รายการคุณสมบัติคร่าวๆ

  • ข้อมูล OHLCV ถูกล้างและบันทึกโดยใช้ Zarr.jl

  • มีวิธีการกรอง คู่ ซึ่งผมเรียกว่าตลาดที่คุณซื้อขายด้วยประกอบด้วยสกุลเงินหลักและสกุลเงินอ้างอิงเช่นBTC/USDT

  • เป็นไปได้ที่จะสุ่มตัวอย่างใหม่ (สุ่มตัวอย่าง) ข้อมูล ohlcv จากกรอบเวลาที่เล็กกว่าเป็นกรอบที่ใหญ่ขึ้น

  • CLI แบบง่ายช่วยให้ดาวน์โหลดและสุ่มตัวอย่างข้อมูลเพื่อให้งาน cron สามารถตั้งค่าให้ดาวน์โหลดข้อมูลเป็นระยะ ๆ

  • มียูทิลิตี้การวางแผนพื้นฐานบางอย่างตาม แผนภูมิ ห้องสมุดผ่าน pycharts เสื้อคลุมหลาม นอกจากนี้ยังมี Echarts.jl แต่มีปัญหาบางอย่างกับมัน ฉันอาจทำงานด้วยการสร้างการกำหนดค่า echarts js ด้วยตัวเองในอนาคตขึ้นอยู่กับว่า pyecharts ดำเนินไปอย่างไร

  • ฉันเล่นเล็กน้อยกับข้อมูลหนังสือสั่งซื้อและความไม่สมดุลของหนังสือสั่งซื้อผ่านโมดูล OrderBook

  • NSAnalysis โมดูลเกี่ยวข้องกับตัวบ่งชี้และการกรองคุณสมบัติแทน สนับสนุนโดยIndicators.jl แพ็คเกจและCausalityTools.jl แพ็คเกจสำหรับความสัมพันธ์

  • ตัวบ่งชี้ตลกอื่น ๆ ที่ฉันเปลี่ยนจาก a หนังสือที่หาได้ง่ายในเว็บ จัดทำโดยโมดูลMVP, Violations สำหรับการลัดวงจรและConsiderations สำหรับความปรารถนา
