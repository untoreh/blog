เหตุใด Windows จึงเป็นภูเขาที่แก้ไขไม่ได้

Windows (11) พูดจาโผงผาง

นี่คือรายการสาเหตุที่หน้าต่างจะไม่มีวันดีไปกว่าการขยับทราย

มันเต็มไปด้วย telemetry

OS ping ปลายทางกี่จุด บ่อยแค่ไหน? สามารถปิดการใช้งาน telemetry ทั้งหมดได้หรือไม่? อาจจะไม่. ระบบปฏิบัติการไม่ได้รับการพัฒนาโดยปิดใช้งานการวัดและส่งข้อมูลทางไกล แต่จะเกิดขึ้นในลักษณะการหมดเวลา / แผงลอยแปลก ๆ เนื่องจากคาดว่าจุดปลาย MS จะเป็น เข้าถึงได้.

รหัสข้อผิดพลาดที่คลุมเครือด้วยสาเหตุที่ไม่ระบุ

Windows ทั้งหมดจะแจ้งให้คุณทราบเมื่อมีบางอย่างผิดพลาดคือรหัสข้อผิดพลาด แทบไม่เคยเป็นข้อความแสดงข้อผิดพลาดเลย และหากเป็นเช่นนั้นก็จะไม่สามารถเข้าใจได้ เนื่องจากจะอ้างอิงรันไทม์บางตัวที่น่าจะปิดแหล่งที่มาได้มากที่สุด

ACL ทุกที่

แน่นอนว่า ACL เป็นรูปแบบการรักษาความปลอดภัยที่ดีกว่า ขอให้โชคดีในการค้นหาตำแหน่งและสิทธิ์ที่จำเป็นเมื่อคุณได้รับข้อผิดพลาดปฏิเสธการอนุญาต

ไดรเวอร์

Windows ไม่รู้จักอุปกรณ์นี้...คุณจะแก้ไขได้อย่างไร ขอจิ้มที่สวยๆ หน่อยได้มั้ยคะ? ฉันไม่ได้พูดถึง ลายเซ็นคนขับ.

สถานะผู้ใช้

ข้อมูลผู้ใช้ถูกเก็บไว้ใน%APPDATA%local และroaming . ถ้าฉันเรียกดูทั้งสองโฟลเดอร์ ฉันไม่เห็นคำจำกัดความที่ชัดเจน แอพบางตัวเก็บแคชไว้ในโฟลเดอร์เดียวแม้ว่าจะเป็นโฟลเดอร์สำหรับ การกำหนดค่า คนอื่นทำตรงกันข้าม...หน้าต่างมีแบบแผนที่เหมาะสมหรือไม่?

Windows ไม่มี PATH

ปกติทุกไบนารีจะถูกเก็บไว้ในไดเร็กทอรีของตัวเอง (พร้อมกับการขึ้นต่อกันของผู้ขายทั้งหมด) หากคุณพยายามสรุปพาธ คุณจะจบลงด้วยรายการพันไดเร็กทอรีที่เก็บไบนารี อย่างน้อยก็มีscoop.

อัปเดต Windows ถูกผลัก...โกรธมาก

เห็นได้ชัดว่าอาจเกิดขึ้นที่ windows ผลักดันการอัปเดตที่ไม่ดี ซึ่งมากกว่าการดึงกลับ หรือการอัปเดตที่ไม่ได้มีไว้สำหรับแพลตฟอร์มที่พวกเขาถูกผลักไป! นอกจากนี้ ยังไม่มีการกล่าวถึงการอัปเกรด windows ว่าเป็นอะตอม ดังนั้นทุกครั้งที่คุณทำการอัพเกรด คุณจะทอยลูกเต๋า

UI

เปลือกหน้าต่างเป็นเหมือนakiosk มากกว่า UI คุณสามารถกดปุ่มและคลิกที่สิ่งต่างๆ ได้ ตราบใดที่ถูกลงโทษโดยทีม windows UI จะไม่มี API ที่เหมาะสมสำหรับทางลัด ทุกแอปพลิเคชันในการกำหนดค่าแป้นพิมพ์ลัดเป็นการแฮ็ก
