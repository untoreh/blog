Por qué Windows es una montaña de errores irreparable

Diatriba de Windows (11).

Aquí hay una lista de razones por las que las ventanas nunca serán mejores que arenas movedizas.

Está lleno de telemetría.

¿A cuántos puntos finales hace ping el sistema operativo? ¿Con que frecuencia? ¿Se puede desactivar la telemetría por completo? Probablemente no. El sistema operativo no está desarrollado con la telemetría deshabilitada, incurrirá en algunos tiempos de espera / comportamientos de bloqueo extraños porque espera que los puntos finales de MS sean accesible.

Códigos de error crípticos con causas no especificadas

Todo lo que Windows le brinda cuando algo sale mal es un código de error, casi nunca un mensaje de error, y si lo hace, es incomprensible porque hace referencia a algún tiempo de ejecución que probablemente sea de código cerrado.

LCA en todas partes

Por supuesto, ACL es el mejor modelo de seguridad. Buena suerte para descubrir dónde y qué permisos se requieren cuando recibe un error de permiso denegado.

Conductores

Windows no ha reconocido este dispositivo... Entonces, ¿cómo solucionas esto? ¿Puedo hurgar en algún lugar bonito, por favor? ni siquiera estoy mencionando firmas del conductor.

Estado de usuario

Los datos del usuario se almacenan en%APPDATA%local yroaming . Si navego por las dos carpetas, no veo definiciones claras, algunas aplicaciones almacenan caché en una carpeta a pesar de que es para configuración otros hacen lo contrario... ¿Tiene Windows las convenciones adecuadas?

Windows no tiene RUTA

Cada binario generalmente se almacena dentro de su propio directorio (junto con todas las dependencias proporcionadas). Si intenta generalizar la ruta, terminaría con una lista de miles de directorios donde se almacenan los archivos binarios, al menos hayscoop.

Las actualizaciones de Windows se envían... con mucha furia

Aparentemente, puede suceder que Windows inserte actualizaciones incorrectas, que luego retroceden, ¡o actualizaciones que ni siquiera son para la plataforma a la que se envían! Tampoco se menciona que las actualizaciones de Windows sean atómicas, por lo que cada vez que realiza una actualización, está tirando los dados.

interfaz de usuario

El caparazón de Windows es más como unkiosk que una interfaz de usuario, puede presionar botones y hacer clic en cosas, siempre que esté autorizado por el equipo de interfaz de usuario de Windows, no existe una API adecuada para los accesos directos. Toda aplicación para configurar atajos de teclado es un hack.
