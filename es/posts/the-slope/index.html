La pendiente

¿Cuánto es demasiado?

Cuando cuantificas cosas, normalmente hay una curva que te dice cuánto necesitas. Esto generalmente se puede trazar de manera que pueda extrapolar un punto de inflexión ideal.

Todos los elementos de esta base de datos deberían tener:

  • una definición parax yy ejes

  • a mínimo y máximo para cada eje

  • una función para trazar la curva (ya sea matemática o ya implementada en un idioma prog.)
