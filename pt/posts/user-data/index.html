Armazene sua atividade na web

Não deixe seus pensamentos em plataformas que você não possui

Quando você posta uma resposta em um site, ou rede social, ela fica lá, talvez por muito tempo, ou eventualmente será deletada. Ter um middleware entre a saída e a entrada que armazena tudo o que você escreve localmente é algo que pode parecer inútil no curto prazo, mas conforme você acumula mais dados, pode ser usado como um banco de dados para uma pesquisa de texto completo de seus pensamentos, ou treinar um bot de bate-papo para dar respostas semelhante ao seu.

O problema com esse software é a interface, o ideal é que você queira armazenar o texto depois de formatado corretamente, por isso deve ser um todo Editor WYSIWYG com o qual os usuários interagem em vez do editor oferecido no site / plataforma. Isso geralmente parece desajeitado e opaco e, considerando que o interesse de curto prazo de registrar todas as suas interações é muito pequeno, uma implementação que seja mais amigável para a adoção do usuário é difícil.

Existem soluções para editores como o emacs / vim que permitem que você escreva dentro do editor e depois envie o texto para o destino, embora neste caso o interesse seja melhorar a experiência de edição, não em manter o controle da saída.
