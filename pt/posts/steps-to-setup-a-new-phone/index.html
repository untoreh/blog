Etapas para configurar um novo telefone

Siga as etapas para limpar um telefone Android padrão

  • Encontre o telefone com base em suas preferências, as ofertas de armazém da Amazon ou o aliexpress remodelado têm o melhor custo-benefício AFAIK.

  • O telefone pode exigir um código de desbloqueio do fabricante

  • Faça check-in fastbootoem unlock se requer um código

  • Considere se um firmware de estoque OEM deve ser atualizado a partir do fastboot antes de atualizar uma ROM personalizada (verifique flashfile.xml)

  • Flash uma recuperaçãofastboot flash recovery twrp.bin

  • Arrancar na ROM flash de recuperação

  • Flash microG por meio do script de configuração não oficial

  • Flash magisk

  • Desinstalador de módulo Magisk para remover módulos que causam bootloops de recuperação

  • Talvez pisque um kernel personalizado

  • Verifique o autoteste microG para ver o que precisa ser configurado (spoofing de apk, nlp unificado etc.)

  • Instale xposed com magisk

  • Configure o lucky patcher

  • Mercados de instalação: f-droid, aurora, aptoide

  • Substituir aplicativos: calendário-> etar, mensagens-> sinal (sessão?), E-mail-> deltachat (ou nada ..), teclado-> anysoftkb, arquivos-> mixplorer, navegador-> bromite / firefox, youtube-> newpipe, câmera-> câmera aberta, música-> vinil / fonógrafo, espaços de trabalho-> abrigo

  • Aplicativos de bateria: melhores estatísticas de bateria, interromper o carregamento da bateria (não deve ser necessário), tornar mais ecológico
