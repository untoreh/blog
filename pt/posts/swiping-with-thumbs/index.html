deslizando com os polegares

Uma interface de teclado de smartphone para deslizar com os polegares

Existem diferentes métodos de entrada de teclado para smartphones:

  • Digitação normal

  • Digitação do polegar

  • Deslizamento normal

  • Digitação compacta (T9)

O deslizamento duplo de entrada foi muito pouco explorado. O único teclado que encontrei para implementar uma boa solução de trabalho é teclado 69.

Por que isso é tão incomum?

  • A implementação é difícil. Entrada múltipla significa que há processos assíncronos preenchendo um buffer de entrada que deve ser consultado para candidatos de conclusão em cada novo evento.

  • Requer algum aprendizado do usuário, o que o torna pouco atraente, pois quanto maior a curva de aprendizado menor a retenção do usuário.
