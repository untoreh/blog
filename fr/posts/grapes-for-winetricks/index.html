raisins pour winetricks

Une configuration basée sur des couches pour les recettes de winetricks

Vin est utilisé pour exécuter des programmes Windows sur Linux. Le runtime fonctionne en réimplémentant dll être compatible Linux. Cependant, le logiciel n'a pas seulement besoin des bibliothèques de base, il existe des dépendances qui doivent être installées pour chaque logiciel.

Wine prend en charge différentes versions de Windows... (toutes ?), il est donc nécessaire d'utiliser différents environnements pour assurer la compatibilité entre le logiciel et les versions de Windows.

Certaines dépendances peuvent également être incompatibles entre elles, c'est pourquoi l'utilisation de préfixes de vin . L'utilisation de préfixes wine signifie que chaque application a généralement son propre environnement (comme les conteneurs), ce qui signifie que toutes les dépendances doit être réinstallé pour chaque nouveau programme . Vous n'en avez pas non plus, différents programmes peuvent partager le même environnement, mais comme indiqué, des problèmes difficiles à déboguer peuvent survenir.

Winetricks automatise l'installation des dépendances (comme les packages .NET et DirectX), mais même si elles sont scriptées, les configurations peuvent encore prendre un certain temps.
