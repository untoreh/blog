Étapes pour configurer un nouveau téléphone

Suivez les étapes pour dégonfler un téléphone Android d'origine

  • Trouvez le téléphone en fonction de vos préférences, les offres d'entrepôt amazon ou aliexpress remis à neuf ont le meilleur rapport qualité-prix AFAIK.

  • Le téléphone peut nécessiter un code de déverrouillage du fabricant

  • Enregistrement rapideoem unlock si cela nécessite un code

  • Considérez si un firmware OEM doit être flashé à partir de fastboot avant de flasher une ROM personnalisée (vérifiez flashfile.xml)

  • Flasher une récupérationfastboot flash recovery twrp.bin

  • Démarrez dans la rom flash de récupération

  • Flash microG via le script de configuration non officiel

  • Magie Flash

  • Programme de désinstallation du module Magisk pour supprimer les modules provoquant des boucles de démarrage de la récupération

  • Peut-être flasher un noyau personnalisé

  • Vérifiez l'autotest microG pour voir ce qui doit être configuré (usurpation d'apk, nlp unifié, etc.)

  • Installer xposed avec magisk

  • Configurer Lucky Patcher

  • Marchés d'installation : f-droid, aurora, aptoide

  • Remplacer les applications : calendrier->etar, messages->signal(session ?), mail->deltachat(ou rien..), clavier->anysoftkb, fichiers->mixplorer, navigateur->bromite/firefox, youtube->newpipe, caméra-> caméra ouverte, musique-> vinyle/phonographe, espaces de travail-> abri

  • Applications de batterie : meilleurs statistiques de batterie, arrêt de la charge de la batterie (ne devrait pas être nécessaire), greenify
