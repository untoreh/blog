Pourquoi Windows est une montagne de bogues incorrigibles

Windows (11) diatribe.

Voici une liste de raisons pour lesquelles les fenêtres ne seront jamais meilleures que les sables mouvants.

C'est plein de télémétrie

Combien de terminaux le système d'exploitation envoie-t-il ? à quelle fréquence? La télémétrie peut-elle être entièrement désactivée ? Probablement pas. Le système d'exploitation n'est pas développé avec la télémétrie désactivée, il entraînera des délais d'attente / blocages étranges car il s'attend à ce que les points de terminaison MS soient accessible.

Codes d'erreur cryptiques avec des causes non spécifiées

Tout ce que Windows vous donne quand quelque chose ne va pas est un code d'erreur, presque jamais un message d'erreur, et si c'est le cas, il est incompréhensible car il fait référence à un runtime qui est très probablement une source fermée.

LCA partout

Bien sûr, ACL est le meilleur modèle de sécurité. Bonne chance pour savoir où et quelles autorisations sont requises lorsque vous obtenez une erreur d'autorisation refusée.

Conducteurs

Windows n'a pas reconnu cet appareil... Alors, comment résoudre ce problème ? Puis-je fouiller quelque part joli s'il vous plaît? je ne mentionne même pas signatures des conducteurs.

État utilisateur

Les données de l'utilisateur sont stockées dans%APPDATA%local etroaming . Si je parcoure les deux dossiers, je ne vois pas de définitions claires, certaines applications stockent le cache dans un dossier même si c'est pour configuration d'autres font le contraire... Les fenêtres ont-elles même des conventions appropriées ?

Windows n'a pas de PATH

Chaque binaire est généralement stocké dans son propre répertoire (avec toutes les dépendances fournies). Si vous essayez de généraliser le chemin, vous vous retrouverez avec une liste de milliers de répertoires où les binaires sont stockés, au moins il y ascoop.

Les mises à jour Windows sont poussées... très en colère

Apparemment, il peut arriver que Windows pousse de mauvaises mises à jour, qui se retirent ensuite, ou des mises à jour qui ne sont même pas destinées à la plate-forme sur laquelle elles sont poussées ! De plus, il n'est pas mentionné que les mises à niveau de Windows sont atomiques, donc chaque fois que vous effectuez une mise à niveau, vous lancez les dés.

interface utilisateur

La coque Windows ressemble plus à unkiosk qu'une interface utilisateur, vous pouvez appuyer sur des boutons et cliquer sur des éléments, tant que cela est sanctionné par l'équipe de l'interface utilisateur Windows, il n'y a pas d'API appropriée pour les raccourcis. Chaque application pour configurer les raccourcis clavier est un hack.
