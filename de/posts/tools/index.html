Aktion5

[Action5] ist ein einfaches Tool, um geänderte Dateien anhand von Prüfsummen in einem Verzeichnis zu erkennen. Es gibt eine Unmenge von Tools, die dies tun, daher bin ich mir nicht sicher (erinnere mich), warum ich es geschrieben habe.

  • über einen Ordner rekursieren, einen Baum von Prüfsummen (den Index) ausgeben..

  • ..wiederholen Sie, um zu sehen, was sich geändert hat

Ein wichtiger Punkt dieses Tools ist, dass es sich nur um Shell-Befehle handelt, sodass Sie jeden beliebigen Prüfsummenbefehl einfügen können, den Sie bevorzugen, der standardmäßig verwendet wirdmd5.

bash-utils

Der Zweck von bashutils besteht darin, sich hauptsächlich auf die bash zu verlassen, um einige Vorgänge an Prozessen durchzuführen.

  • Lesen einer Datei in einer Variablen (gobbler ) erfordert, dass die Datei keine Nullen enthält, daher muss sie mit einem Null-freien Kodierungsformat gespeichert werden. Hinweis: Die unbeschwerte Verwendung von Anführungszeichen in der bash (an Stellen, an denen es nicht erforderlich ist) kann dazu führen, dass viel Speicher verschwendet wird.

  • Überwachen der CPU in einem Linux-System, muss gelesen werden/proc , pausieren Sie ohne Forking und berechnen Sie Statistiken nur mit Shell-Operationen ($(()))

  • Die Begrenzung eines Prozesses erfordert kontinuierliches SendenSIGSTOP undSIGSTOP Signale, dies ist in bash sehr langsam :) da es sich um eine Schleife handelt, und Software kann sich unerwartet verhalten, wenn sie ständig angehalten und fortgesetzt wird, da Programmierer normalerweise nicht erwarten, dass ihre Software zeitweilig arbeitet!
