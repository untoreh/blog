Speichern Sie Ihre Webaktivitäten

Hinterlasse deine Gedanken nicht auf Plattformen, die du nicht besitzt

Wenn Sie eine Antwort auf einer Website oder einem sozialen Netzwerk posten, bleibt sie möglicherweise für lange Zeit dort oder wird schließlich gelöscht. Eine Middleware zwischen Ausgabe und Eingabe zu haben, die alles speichert, was Sie lokal schreiben, kann auf kurze Sicht nutzlos erscheinen, aber wenn Sie mehr Daten ansammeln, kann sie als Datenbank für eine Volltextsuche Ihrer Gedanken verwendet werden, oder einen Chatbot trainieren, Antworten zu geben ähnlich wie bei dir.

Das Problem bei solcher Software ist die Benutzeroberfläche, idealerweise möchten Sie den Text speichern, nachdem er richtig formatiert wurde, also sollte er a ganz WYSIWYG-Editor, mit dem Benutzer anstelle des auf der Website/Plattform angebotenen Editors interagieren. Dies fühlt sich normalerweise klobig und undurchsichtig an, und wenn man bedenkt, dass das kurzfristige Interesse, all Ihre Interaktionen zu protokollieren, sehr gering ist, ist eine Implementierung, die für die Benutzerakzeptanz freundlicher ist, schwierig.

Es gibt Lösungen für Editoren wie emacs/vim, die es Ihnen ermöglichen, innerhalb des Editors zu schreiben und dann den Text an das Ziel zu senden, obwohl in diesem Fall das Interesse an der Verbesserung der Bearbeitungserfahrung besteht und nicht daran, die Ausgabe zu verfolgen.
