Macht der Menge

Apps mit Crowdsourcing

Dies ist eine Liste von Apps, die die gleiche allgemeine Idee haben, Benutzereingaben und soziale Interaktionen zu nutzen.

Soziales Gemüse

Es gibt einige Machbarkeitsnachweise über ein soziales Netzwerk, das sich auf den Anbau und den Austausch von Gemüse konzentriert. Aber es ist nichts Ernsthaftes entstanden. Es macht nur auf Amateurebene Sinn, gleichzeitig kann es nur an Orten mit einer hohen Dichte an grüngetümmelten Menschen abheben, so dass ein Netzwerk entstehen kann; Züchter müssen auch ein Vertrauensnetz aufbauen, sonst können keine Transaktionen stattfinden.

Müllsucher

Bewerten Sie, wie viel ein bestimmter Ort mit Müll gefüllt ist. Der Benutzer kann mit Geo-Tags versehene Fotos einreichen, die mit Müll gefüllt sind, und eine Bewertung von bis zu 3 Sternen geben, wie schlecht es ist. Dies kann den Gemeinden helfen, den Umweltzustand ihrer Territorien zu verfolgen, aber es fehlt an Anreizen.

Lochsucher

Wie Müllsucher, aber für Straßenlöcher. Dies kann durch die Verwendung von Smartphone-Sensoren verbessert werden, um ungewöhnliche plötzliche Bewegungen zu erkennen das darf nicht passieren wenn auf der Straße.

Gedränge

Bevölkerungsdichte aus Sicht des menschlichen Auges. Nützlich, wenn Sie verfolgen möchten, wie überfüllt Orte wie Strände oder Parks sind, damit Sie sich für weniger überfüllte Ziele entscheiden können.

Standpunkt

Eine pseudokritische/satirische Website/Datenbank/Wörterbuch, auf der Dinge auf unterschiedliche Weise definiert werden, wie sie von einer bestimmten stereotypen Persönlichkeit/Charakter gesehen werden.
