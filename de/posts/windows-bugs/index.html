Warum Windows ein nicht behebbarer Berg von Fehlern ist

Windows (11) schimpft.

Hier ist eine Liste mit Gründen, warum Fenster niemals besser sein werden als Flugsand.

Es ist voller Telemetrie

Wie viele Endpunkte pingt das Betriebssystem? wie häufig? Kann die Telemetrie vollständig deaktiviert werden? Wahrscheinlich nicht. Das Betriebssystem wird nicht mit deaktivierter Telemetrie entwickelt, es kommt zu einigen seltsamen Zeitüberschreitungen / Verzögerungen, da es MS-Endpunkte erwartet erreichbar.

Kryptische Fehlercodes mit unbestimmten Ursachen

Alles, was Windows Ihnen gibt, wenn etwas schief geht, ist ein Fehlercode, fast nie eine Fehlermeldung, und wenn doch, ist es unverständlich, weil es auf eine Laufzeit verweist, die höchstwahrscheinlich Closed Source ist.

ACL überall

Natürlich ist ACL das bessere Sicherheitsmodell. Viel Glück beim Herausfinden, wo und welche Berechtigungen erforderlich sind, wenn der Fehler „Berechtigung verweigert“ angezeigt wird.

Treiber

Windows hat dieses Gerät nicht erkannt ... Wie können Sie das beheben? Kann ich bitte irgendwo hübsch stochern? Ich erwähne es nicht einmal Fahrersignaturen.

Benutzerstatus

Benutzerdaten werden gespeichert in%APPDATA%local undroaming . Wenn ich die beiden Ordner durchsuche, sehe ich keine klaren Definitionen, einige Apps speichern den Cache in einem Ordner, obwohl es dafür vorgesehen ist Aufbau andere machen das Gegenteil ... Hat Windows überhaupt richtige Konventionen?

Windows hat keinen PATH

Jede Binärdatei wird normalerweise in einem eigenen Verzeichnis gespeichert (zusammen mit allen vom Hersteller bereitgestellten Abhängigkeiten). Wenn Sie versuchen, den Pfad zu verallgemeinern, würden Sie am Ende eine Liste mit tausend Verzeichnissen haben, in denen Binärdateien gespeichert sind, zumindest gibt es einescoop.

Windows-Updates werden gepusht ... sehr wütend

Anscheinend kann es vorkommen, dass Windows fehlerhafte Updates pusht, die dann zurückgezogen werden, oder Updates, die nicht einmal für die Plattform sind, auf die sie gepusht werden! Es wird auch nicht erwähnt, dass Windows-Upgrades atomar sind, also würfeln Sie jedes Mal, wenn Sie ein Upgrade durchführen.

Benutzeroberfläche

Die Windows-Shell ist eher wie einekiosk Als eine Benutzeroberfläche können Sie Schaltflächen drücken und auf Dinge klicken, solange dies vom Windows-Benutzeroberflächenteam genehmigt wird, gibt es keine geeignete API für Verknüpfungen. Jede Anwendung zum Konfigurieren von Tastaturkürzeln ist ein Hack.
