Julia 回测框架

与 Julia 交易加密货币。

回测.jl 是我写的一个 julia 项目，用于处理加密货币交易。目前它与 ccxt 用于连接交易所 api 的 python 库，现在只是为了获取 ohlcv 数据。

功能的粗略列表

  • OHLCV 数据使用 扎尔.jl

  • 有过滤方法 对 这就是我所说的你交易的市场，由基础货币和报价货币组成，比如BTC/USDT

  • 可以将 ohlcv 数据从较小的时间帧重新采样（下采样）到较大的时间帧。

  • 一个简单的 CLI 允许下载和重新采样数据，以便可以设置 cron 作业以定期下载数据。

  • 有一些基于 图表 图书馆通过 pycharts 蟒蛇包装。还有 Echarts.jl 但有一些问题。未来我可能会根据 pyecharts 的进展情况来创建自己的 echarts js 配置。

  • 我通过 OrderBook 模块对一些订单簿数据和订单簿不平衡进行了一些处理。

  • 这Analysis 相反，模块处理指标和特征过滤。它由Indicators.jl 包，以及CausalityTools.jl 相关性包。

  • 其他有趣的指标，我从 您可以在网上轻松找到的书 由模块提供MVP, Violations 对于短路和Considerations 因为渴望。
