关于通货膨胀

是什么导致了资本主义市场的通货膨胀。

我们现在更好地理解了我们对通货膨胀知之甚少

随着新冠疫苗分发后行业恢复到完整状态（或接近状态），预计商品价格将恢复原状。

做出这种假设的人只读过市场，从未参与过市场。

市场规则

  • 没有利润就没有市场。这是基本假设，没有人免费工作。这听起来可能很明显，但我相信存在如此脱离现实的人认为人们“只是生存”可能很好。不，“只是幸存”部分只会让您达到“没有反抗”的程度。

  • 如果你能赚到更多的利润，你就这么做。

结果

在你进入失落期之后会发生什么？你想弥补这些损失，所以你提高了价格。在您提高价格以重新建立利润之后……为什么要降低价格？

inertia + profits == Inflation

...但是等等...还有更多！

供应链多米诺骨牌

涨价是 不是 一个时间点。从供应链中的一个参与者提高价格会导致多米诺骨牌效应，每个下游节点要么承担成本，要么将成本推到较低的节点[1] ，直到最后一个节点（消费者）。不要混淆 无孔不入 例如能源价格上涨，同时影响所有节点。

sum(loss(node) for node in source..consumer) == Inflation

自我充实的预言

通胀预期会引发通胀。如果媒体谈论“可能的”通货膨胀，它就会发生，因为每个人都会试图 为它做准备 提前通过......提高价格并获得更多利润。

loss(t+1) -> Inflation(t)

暴徒

当一群人提高商品价格时会发生什么？其他参与者可能会通过同伴效应做同样的事情。 自发的卡特尔 出生！不是任何人真正创造的，只是外力造成的，只有其他同样强大的外力才能拆除。

(profits(b) + 1 -> profits(B - {b}) + 1 for every b in B) -> inflation(B)

竞争在哪里！？

竞争是 不是给定的.

  • 新兴市场的竞争（更）激烈，而所有成熟市场最终都会变成垄断或双头垄断。停滞不前的经济（例如后疫情时代的经济）拥有较少的新兴市场。

  • 一些市场需要太多的前期资金来吸引竞争。

  • 竞争通常来自对利润的合理预期（例如，由于更好的技术、物流等），而不是来自现有企业的任意错误定价。

QE/QT 如何适应这一点？

有一天，我希望了解控制一个经济体的货币供应应该如何“解决问题”。对我来说，它看起来像：

  • 紧急按钮 : 当你不知道该怎么做时，发怒[2] 钱，它应该做点什么！

  • 锤子 : 每一个经济问题，都是一个钉子。

无论如何，为了应对通货膨胀，中央银行（害怕货币贬值）决定 QT.这意味着流动性减少，贷款减少，筹集的资金减少，随之而来的是衰退。由于现金流量减少，市场竞争减少，现有企业没有理由降低价格，通胀仍然不受影响。 QT，以捍卫货币价值为目标，有 翻倍 通货膨胀的肇事者赚取的额外利润。

[1]至此，您应该熟悉 市场规则
[2]你可能把它误读为“抽奖”，我也没问题。
