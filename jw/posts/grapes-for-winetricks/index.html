anggur kanggo winetricks

Persiyapan adhedhasar lapisan kanggo resep winetricks

Anggur digunakake kanggo mbukak program windows ing linux. Runtime bisa digunakake maneh dlls dadi kompatibel karo linux. Nanging piranti lunak ora mung mbutuhake perpustakaan dhasar, ana katergantungan sing kudu diinstal kanggo saben piranti lunak.

Anggur ndhukung macem-macem versi windows ... (kabeh mau?), Mula prelu nggunakake macem-macem lingkungan kanggo njamin kompatibilitas antarane versi piranti lunak lan windows.

Sawetara dependensi bisa uga ora kompatibel karo siji liyane, mula panggunaan awalan anggur . Nggunakake awalan anggur tegese biasane saben aplikasi duwe lingkungan dhewe, (kaya kontainer), tegese kabeh katergantungan kudu diinstal maneh kanggo saben program anyar . Sampeyan ora duwe, macem-macem program bisa nuduhake lingkungan sing padha, nanging kaya sing wis ditemtokake, bisa uga ana masalah sing angel debug.

Winetricks otomatis nginstal katergantungan (kaya .NET lan paket DirectX), sanajan skrip, persiyapan isih bisa suwe.

  • Proposal

Ostree digunakake kanggo checkout versi saka sistem operasi . Lan sanajan kasus panggunaan utamane yaiku ngaktifake upgrade atom kanggo OS sing dipasang ing logam kosong, bisa digunakake kanggo mbangun awalan anggur kanthi koleksi dependensi sing wis diinstal, kaya flatpak nggunakake cara sing padha kanggo nuduhake runtime ing antarane aplikasi sing beda, kanthi efektif bisa uga ana flatpak sing beda runtime anggurkanggo kulawarga piranti lunak sing nuduhake akeh katergantungan; mung pitakonan sing isih ana yaiku yen ana masalah lisensi sing nyebarke wektu runtime kaya apa sing bakal efektif piranti lunak windows sing wis diinstal . : S
