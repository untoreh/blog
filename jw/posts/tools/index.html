tumindak5

[Action5] minangka alat balung gundhul kanggo nemokake file sing diowahi liwat checkum ing direktori. Ana alat gazillion sing nindakake iki, mula aku ora yakin (ngelingi) kenapa aku nulis.

  • recurse liwat folder, mbuwang wit checkums (indeks) ..

  • ..baleni maneh kanggo ndeleng apa sing owah

Titik utama alat iki yaiku mung printah cangkang, dadi sampeyan bisa masang perintah pamriksan sing disenengi, kanthi standar nggunakakemd5.

bash-utils

Tujuane saka basutil yaiku umume gumantung karo bash kanggo nindakake operasi ing proses.

  • Maca file ing njero variabel (gobbler ), kudu priksa manawa file ora ngemot nulls, mula file kasebut kudu disimpen nganggo format enkoding gratis sing nol. Cathetan: panggunaan kuotasi sing ora preduli ing bash, (ing papan sing ora dibutuhake) bisa nyebabake akeh memori sing boros.

  • Ngawasi CPU ing sistem linux, kudu diwaca/proc , ngaso tanpa forking, lan ngetung statistik kanthi mung operasi shell ($(()))

  • Matesi proses kudu terus dikirimSIGSTOP lanSIGSTOP sinyal, iki alon banget ing bash :) amarga iku daur ulang, lan piranti lunak bisa uga tumindak ora diduga nalika terus mandheg lan diterusake maneh, amarga programer ora biasane ngarepake piranti lunak kasebut bisa digunakake kanthi intermiten!

dist-haproxy

A skrip mbangun kanggo haproxy , alat sing paling apik kanggo nerusake koneksi, entheng, lan duwe versi sing disambung kanthi statis migunani banget.
