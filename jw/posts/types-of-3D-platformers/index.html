jinis platformer 3D

jinis game apa sing bisa diarani platformer?

Platformer 3D minangka jinis game sing duwe kamera lan tujuan utamane yaiku pindhah saka titik A menyang B.

Fitur umum

  • papriksan : saben level diisi karo titik centhang, platforming yaiku nglakokake urutan tumindak (masukan), kesalahan umume, lan asile biasane game liwat . Checkpoints nyuda rasa nyeri sing paling umum saka platformer ... wiwit saka awal.

  • speedruns: platformer minangka game sing paling umum kanggo mlayu kanthi cepet, amarga mbutuhake tliti ing saben langkah, padha karo muter alat musik, yen sampeyan gagal nglakokake kombinasi sing bener, asile ora nyenengake.

  • peta lan addons : ana macem-macem macem-macem peta amarga peta utamane isi game kasebut, komunitas uga dikembangake kanggo nggawe lan nyoba karo macem-macem tata letak peta lan jinis alangan ing njeronÃ©.

  • bebarengan lan kompetisi : kompetisi kalorone ana ing bentuk papan pamimpin lan PvP gaya balapan cocog. Nanging, amarga langit-langit keterampilan sing luwih dhuwur, lan risiko gagal umum ing eksekusi, papan peringkat utama minangka gerbang utama kanggo kompetisi.

Sub kategori

Platformer bisa dipisahake adhedhasar jinis fisika sing digunakake kanggo nggawe gerakan:

  • Pelancar : gliding, drifting, landai lan lereng ngontrol arah PG, sing biasane gerakan terus-terusan wiwit wiwitan nganti pungkasan, cepet banget.

  • Grapplers : gunakake proyektil, momentum sudut lan pendakian kanggo pindhah ing platform, peta biasane digawe saka langkah-langkah pamriksan biner, sing saben langkah minangka urutan tumindak; amarga iki, pamindhahan PG alon kanthi asring interupsi. Umume golek cangkriman ing grapplers.

  • Parkour : jump-double, jump-bunping, air-control, roket-jumps minangka kemampuan umum sing diwenehake marang PG ing platformer kasebut. Umume umume dicampur FPS unsur kanthi unsur parkour kanthi tujuan eksploitasi gaman kanggo ngatasi alangan.
