Napa Windows minangka gunung bug sing ora bisa didandani

Windows (11).

Iki minangka dhaptar alasan kenapa jendhela ora bakal luwih apik tinimbang ngganti pasir.

Iku kebak telemetri

Pira titik pungkasan OS ping? sepira kerepe? Apa telemetri bisa dipateni kabeh? Mbokmenawa ora. OS ora dikembangake kanthi telemetri dipatèni, bakal ngalami sawetara wektu entek / kios sing aneh amarga ngarepake titik pungkasan MS dadi tekan.

Kode kesalahan samar kanthi panyebab sing ora ditemtokake

Kabeh Windows menehi nalika soko dadi awry kode kesalahan, meh tau pesen kesalahan, lan yen mengkono iku ora dingerteni amarga referensi sawetara runtime sing paling kamungkinan ditutup sumber.

ACL nang endi wae

Mesthine, ACL minangka model keamanan sing luwih apik. Sugeng nemokake ing endi lan ijin apa sing dibutuhake nalika entuk ijin sing ditolak kesalahan.

Supir

Windows durung ngenali piranti iki...Dadi kepiye carane ndandani iki? Bisa aku poke nang endi wae cantik please? Aku malah ora sebutno teken driver.

Negara pangguna

Data pangguna disimpen ing%APPDATA%local lanroaming . Yen aku nelusuri rong folder, aku ora ndeleng definisi sing jelas, sawetara aplikasi nyimpen cache ing siji folder sanajan iku kanggo konfigurasi liyane nindakake ngelawan ... Apa windows malah duwe konvènsi tepat?

Windows ora duwe PATH

Saben binar biasane disimpen ing direktori dhewe (bebarengan karo kabeh dependensi sing didol). Yen sampeyan nyoba kanggo generalize path sampeyan bakal entuk dhaptar ewu direktori ing ngendi binari disimpen, paling ora anascoop.

Nganyari Windows di-push ... nesu banget

Ketoke bisa kelakon yen windows nyurung nganyari ala, kang saka narik bali, utawa nganyari sing ora malah kanggo platform sing di-push kanggo! Uga ora ana sebutno upgrade windows sing atom, supaya saben-saben sampeyan nindakake upgrade, sampeyan muter dadu.

UI

Cangkang windows luwih kaya akiosk tinimbang UI, sampeyan bisa menet tombol lan ngeklik barang, anggere disetujoni dening tim UI windows, ora ana API sing cocog kanggo trabasan. Saben aplikasi kanggo ngatur trabasan keyboard minangka hack.
