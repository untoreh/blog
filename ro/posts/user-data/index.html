Stocați-vă activitatea web

Nu vă lăsați gândurile pe platforme pe care nu le dețineți

Când postați un răspuns pe un site web sau pe o rețea socială, acesta este lăsat acolo, poate pentru o lungă perioadă de timp, sau în cele din urmă va fi șters. A avea un middleware între ieșire și intrare care stochează orice scrieți local este ceva ce poate părea inutil pe termen scurt, dar pe măsură ce acumulați mai multe date, poate fi folosit ca bază de date pentru o căutare completă a textului în gândurile voastre sau pentru a instrui un bot de chat pentru a da răspunsuri asemănător cu al tău.

Problema cu un astfel de software este interfața, în mod ideal doriți să stocați textul după ce este formatat corect, deci ar trebui să fie un întreg Editor WYSIWYG cu care utilizatorii interacționează în locul editorului oferit pe site / platformă. Acest lucru se simte de obicei greșit și opac și având în vedere că interesul pe termen scurt de a înregistra toată interacțiunea dvs. este foarte mic, o implementare care este mai prietenoasă cu adoptarea utilizatorilor este dificilă.

Există soluții pentru editori precum emacs / vim care vă permit să scrieți în editor și apoi să trimiteți textul la destinație, deși în acest caz interesul este îmbunătățirea experienței de editare, nu urmărirea rezultatului.
