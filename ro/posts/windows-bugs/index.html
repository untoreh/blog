De ce Windows este un munte de erori de nereparat

Ferestre (11) rătăcire.

Iată o listă de motive pentru care ferestrele nu vor fi niciodată mai bune decât nisipurile mișcătoare.

Este plin de telemetrie

Câte puncte finale face ping sistemul de operare? cat de des? Telemetria poate fi dezactivată complet? Probabil ca nu. Sistemul de operare nu este dezvoltat cu telemetria dezactivată, va avea un comportament ciudat de timeout-uri / blocaje, deoarece se așteaptă ca punctele finale MS să fie accesibil.

Coduri de eroare criptice cu cauze nespecificate

Tot ce vă oferă Windows atunci când ceva merge prost este un cod de eroare, aproape niciodată un mesaj de eroare și, dacă o face, este de neînțeles deoarece face referire la un timp de execuție care este cel mai probabil sursă închisă.

ACL peste tot

Desigur, ACL este cel mai bun model de securitate. Noroc să afli unde și ce permisiuni sunt necesare atunci când primești o eroare de refuzare a permisiunii.

Șoferii

Windows nu a recunoscut acest dispozitiv... Deci cum remediați acest lucru? Pot să bag undeva destul de te rog? nici nu mentionez semnăturile șoferului.

Statul utilizatorului

Datele utilizatorului sunt stocate în%APPDATA%local șiroaming . Dacă răsfoiesc cele două foldere, nu văd definiții clare, unele aplicații stochează memoria cache într-un singur folder, deși este pentru configurare alții fac contrariul... Windows are chiar convenții adecvate?

Windows nu are PATH

Fiecare binar este de obicei stocat în propriul director (împreună cu toate dependențele vândute). Dacă încercați să generalizați calea, veți ajunge la o listă de mii de directoare în care sunt stocate binare, cel puțin existăscoop.

Actualizările Windows sunt impuse... foarte supărat

Se pare că se poate întâmpla ca Windows să împingă actualizări proaste, care apoi să le retragă sau actualizări care nu sunt nici măcar pentru platforma către care sunt împinse! De asemenea, nu se menționează că upgrade-urile Windows sunt atomice, așa că de fiecare dată când efectuați o actualizare, aruncați zarurile.

UI

Învelișul ferestrei seamănă mai mult cu unkiosk decât o interfață de utilizare puteți apăsa butoane și faceți clic pe chestii, atâta timp cât este sancționat de echipa de interfață de utilizare Windows, nu există un API adecvat pentru comenzi rapide. Fiecare aplicație pentru a configura comenzile rapide de la tastatură este un hack.
