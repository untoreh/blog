виноград для винетриксов

Настройка на основе слоев для рецептов winetricks

Вино используется для запуска программ Windows в Linux. Среда выполнения работает путем повторной реализации dll быть совместимым с Linux. Однако программному обеспечению нужны не только базовые библиотеки, но и зависимости, которые необходимо установить для каждой части программного обеспечения.

Wine поддерживает разные версии Windows ... (все?), Поэтому необходимо использовать разные среды для обеспечения совместимости между версиями программного обеспечения и Windows.

Некоторые зависимости также могут быть несовместимы друг с другом, поэтому использование винные приставки . Использование префиксов Wine означает, что обычно каждое приложение имеет свою собственную среду (например, контейнеры), что означает, что все зависимости необходимо переустановить для каждой новой программы . У вас их тоже нет, разные программы могут использовать одну и ту же среду, но, как уже говорилось, могут возникнуть проблемы, которые трудно отладить.

Winetricks автоматизирует установку зависимостей (например, пакетов .NET и DirectX), но даже при наличии сценария установка может занять некоторое время.

  • Предложение

Острее используется для оформления заказа версии из оперативные системы . И хотя его основной вариант использования - включение атомарных обновлений для ОС, установленных на голом железе, его можно использовать для сборки винные приставки с предустановленными наборами зависимостей, например, Flatpak использует один и тот же метод для разделения времени выполнения между разными приложениями, фактически могут быть разные Flatpak время выполнения виндля семейств программного обеспечения, которые разделяют большую часть зависимостей; остается единственный вопрос, есть ли какие-либо проблемы с лицензией, распространяющие такие среды выполнения с тем, что было бы эффективно предустановленное программное обеспечение Windows . : S
