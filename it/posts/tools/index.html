Strumenti minori

Raccolta di piccole utilità fatte per grattare qualche prurito...

azione5

[Azione5] è uno strumento essenziale per individuare i file modificati tramite i checksum in una directory. Ci sono un trilione di strumenti che lo fanno, quindi non sono sicuro (ricordo) del motivo per cui l'ho scritto.

  • ricorsivo su una cartella, scarica un albero di checksum (l'indice)..

  • ..ripeti per vedere cosa è cambiato

Un punto chiave di questi strumenti è che sono solo comandi di shell, quindi puoi collegare qualsiasi comando di somma di controllo che preferisci, per impostazione predefinita utilizzamd5.

bash-utils

Lo scopo di bashutils è fare affidamento principalmente su bash per eseguire alcune operazioni sui processi.

  • Lettura di un file all'interno di una variabile (gobbler ), richiede di assicurarsi che il file non contenga null, quindi deve essere archiviato con un formato di codifica libero null. Nota: l'uso spensierato delle virgolette in bash (in luoghi dove non richiesto) può causare lo spreco di molta memoria.

  • Il monitoraggio della CPU in un sistema Linux richiede la lettura/proc , metti in pausa senza fork e calcola le statistiche con solo operazioni di shell ($(()))

  • La limitazione di un processo richiede l'invio continuoSIGSTOP eSIGSTOP segnali, questo è molto lento in bash :) poiché è un ciclo e il software potrebbe comportarsi in modo imprevisto quando viene costantemente messo in pausa e ripreso, poiché i programmatori di solito non si aspettano che il loro software funzioni a intermittenza!
