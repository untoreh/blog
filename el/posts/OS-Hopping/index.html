OS Hopping

Μια σύντομη περίληψη των λειτουργικών συστημάτων που έχω χρησιμοποιήσει

Όταν πήρα τον πρώτο μου προσωπικό υπολογιστή, ήρθε με Windows XP (13 ετών). Μετά από μερικές αταξίες έπρεπε να μάθω πώς να επανεγκαταστήσω τα παράθυρα ... Δεν ήταν τόσο plug and play όσο σήμερα ... Η πρώτη μου εμπειρία με το linux ήταν με το PCLinuxOS, το οποίο προφανώς είναι συνεχίζεται καθώς προσέφερε μια πλήρη γραφική εγκατάσταση. Μετά από αυτό μπήκα κατευθείαν Arch Linux Το Έπαιξα με το Arch Linux για λίγο (όπως το να κάνω το gnome UI να μοιάζει με MacOS ή Windows Vista μέσω gnome look mods ), μετά επέστρεψα στα Windows επειδή τα παιχνίδια .. Από τα Windows διαχειρίστηκα μερικούς προσωπικούς διακομιστές linux για λίγο τρέχοντας centos/debian. Μετά από μερικά χρόνια επέστρεψα στο Linux, πρώτα δοκίμασα αλπικός , αλλά τα αποθετήρια του δεν ήταν (ακόμα;) ικανοποιητικά για μια εμπειρία επιφάνειας εργασίας (το pulseaudio ήταν ακόμα στο δοκιμή repo).

Μετά προσπάθησα Κενό Linux για λίγο, αλλά χρησιμοποιούσα το musl βασισμένη έκδοση που δεν είχε καλά πακέτα υποστήριξης, οπότε έτρεχα void linux plus εφαρμογές σε κοντέινερ σε a i3wm βασισμένο περιβάλλον. Όμως, οι εφαρμογές γραφικών με εμπορευματοκιβώτια δεν ήταν ακόμη ευρέως διαδεδομένες, λύσεις όπως το flatpak snapd ή το firejail δεν ήταν ακόμη ευρέως διαδεδομένες ή ακόμη και σταθερές, οπότε άλλαξα στο Ubuntu και έστησα ένα ZFS RAID-Z ( ζουν επικίνδυνα ) εγκατάσταση ρίζας. Ως περιβάλλοντα επιφάνειας εργασίας κράτησα ακόμα το i3wm και μετά πήγα στο swaywm προκαλεί wayland, και στη συνέχεια πλάσμα KDE επειδή δεν μπορούσα να ασχοληθώ άλλο με τα απορρίμματα αρχείων μονάδας συστήματος που πρέπει να χρήστης κατάφερε να πάρει την ελάχιστη αναμενόμενη εργασιακή ρύθμιση με οποιοδήποτε από τα πλακάκια WM.

Το KDE είναι υπέροχο και το θεωρώ το αιμορραγικό άκρο των περιβαλλόντων επιφάνειας εργασίας, ωστόσο είναι επίσης μεγάλο και με πολλές εφαρμογές που στερούνται καλής συντήρησης, οπότε η συμβουλή μου είναι να μην προσπαθήσετε πολύ να κολλήσετε σε περιβάλλον GUI μόνο K* και Απλώς χρησιμοποιήστε αυτό που λειτουργεί, το KDE έχει καλή υποστήριξη για το στυλ GTK. Το KWin έχει πολύ καλές επιλογές προσαρμογής και μπορείτε να συνδέσετε τα πλήκτρα πρόσβασης για να το κάνετε να συμπεριφέρεται πολύ κοντά σε float-first πλακάκια WM. Αν θέλετε α κεραμίδι-πρώτα πλακάκια WM τότε υπάρχουν μερικές επεκτάσεις, οι οποίες ωστόσο δεν λειτουργούν αρκετά καλά στο wayland (τη στιγμή της γραφής).

Τον τελευταίο καιρό επέστρεψα στα Windows (10) .. πάλι , όχι πρωτίστως γιατί παιχνίδια καθώς πρέπει ακόμα να βρω χρόνο και κίνητρο για να παίξω ξανά, αλλά κάπως από περιέργεια για α WSL περιβάλλον Linux βασισμένο σε συνδυασμό με καλή ενσωμάτωση για γραφικές εφαρμογές ( WSLg). Κατά τη στιγμή της γραφής, δυστυχώς η εμπειρία των Windows 10 είναι μια de facto υποβάθμιση από το Linux KDE. Windows με τα ποντικοκεντρικό Το UX δεν φαίνεται να έχει κάτι προσανατολισμένο στην προσαρμογή βαθιών συντομεύσεων πληκτρολογίου. Δεν είναι σαφές εάν έχει περάσει Πολιτισμός των Windows οι συντομεύσεις συνοφρυώνονται, θεωρείται ως μια τρύπα ασφαλείας ή είναι απλώς ένας σοβαρός περιορισμός του κελύφους των παραθύρων, που αναγκάζει τις προσαρμογές μέσω παρακάμψεων μητρώου. Powertoys έχει ακόμη πολύ δρόμο να διανύσει για να ικανοποιήσει όλες τις ανάγκες ενός ατόμου που θέλει να προσαρμόσει το περιβάλλον του στις συνήθειές του (δεν είμαι σίγουρος ότι μπορώ να θεωρηθώ ως χρήστης ενέργειας). Παρ 'όλα αυτά, μπορώ να είμαι αρκετά ευέλικτος και με μερικά προγράμματα τρίτων το περιβάλλον έχει γίνει κάπως αποδεκτό.
