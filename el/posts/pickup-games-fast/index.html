παιχνίδια παραλαβής γρήγορα ... ή PUF

Πώς να μάθετε γρήγορα παιχνίδια;

Μια προσπάθεια δημιουργίας μιας λίστας με κουκκίδες για να μάθετε να παίζετε ένα παιχνίδι ή να μάθετε να παρακολουθείτε ένα παιχνίδι.

Μαθαίνοντας να παρακολουθεί ένα παιχνίδι είναι διαφορετικό γιατί δεν είναι σημαντικό να είσαι καλός σε κάτι για να μπορείς να καταλάβεις τι συμβαίνει. αν και οι μηχανικές δεξιότητες μπορούν πάντα να εκτιμηθούν (και είναι αυτά που δημιουργούν διαφημίσεις ), αυτό που συνήθως γοητεύει τους θαυμαστές είναι το διαδικασίες σκέψης πίσω από τις αποφάσεις των αγώνων.

Μια λίστα για να μάθετε πώς να παίζετε

Τα κουκκίδες πρέπει να εστιάζονται γύρω από τη ροή του παιχνιδιού, πώς να επιτύχετε το Y κάνοντας το x. Η ροή είναι το πιο σημαντικό πράγμα που πρέπει να κατανοήσουμε για ένα παιχνίδι, η μηχανική και η γνώση περιστρέφονται γύρω από αυτό, μπορεί να περιγραφεί ως ο ρυθμός προς την ομαδοποίηση του τελικού στόχου των πιο συνηθισμένων ακολουθιών εισροών.

Χρησιμοποίησα τη λέξη βήμα για να πω ότι η ροή του παιχνιδιού μοιάζει πολύ με ρυθμό. Η κατανόηση του πώς εξελίσσεται ένας αγώνας βιντεοπαιχνιδιών είναι κάπως παρόμοια με την ανάγνωση ενός μουσικού φύλλου, υπάρχουν σκαμπανεβάσματα, γρήγορες στιγμές, αργές στιγμές, κρεσέντο και το περιβόητο η ηρεμία πριν την καταιγίδα.

Μια λίστα για να μάθετε πώς να παρακολουθείτε

Οι κουκκίδες θα πρέπει να εστιάζουν σε συγκεκριμένες πτυχές του παιχνιδιού, μερικές είναι περισσότερο προσανατολισμένες στο είδος, κάποιες στο παιχνίδι. Παράδειγμα, για να εκτιμήσουμε τις συγκρούσεις DOTA, αυτό που είναι σημαντικό δεν είναι τόσο η ίδια η ομάδα, αλλά η θέση των παικτών πριν τον αγώνα της ομάδας. Αυτές οι πληροφορίες μπορούν πραγματικά να εκτιμηθούν και σας κάνουν να αγαπάτε μια ομάδα περισσότερο από μια άλλη, επειδή απεικονίζει την προετοιμασία και τον συντονισμό που περνά ο κάθε παίκτης. Το αποτέλεσμα ενός ομαδικού αγώνα συχνά αποφασίζεται από τη θέση του κάθε παίκτη πριν από αυτό.

Στην αντίθετη απεργία, ο συντονισμός και η εκτέλεση των ρυθμίσεων είναι αυτό που μπορεί να εκτιμηθεί, πολύ περισσότερο από τις φάρσες που τείνουν να είναι επαναλαμβανόμενες στην εκτέλεσή τους και αναμένονται σε ένα ορισμένο επίπεδο δεξιοτήτων. Τα σενάρια 1vX μπορεί επίσης να είναι πραγματικά αξιοθαύμαστα επειδή ξεφεύγουν από τις στρατηγικές της ομάδας όπου τα 2+vX τείνουν να είναι απλώς ένα υποσύνολο αυτών και αφήνουν τον παίκτη να αναδείξει την ικανότητά του και το επίπεδο γρήγορης σκέψης, επίσης τέτοια σενάρια τείνουν να είναι τα πιο πρωτότυπα όσον αφορά τις θέσεις στο χάρτη.
