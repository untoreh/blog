σύροντας με αντίχειρες

Διεπαφή πληκτρολογίου smartphone για σάρωση με αντίχειρες

Υπάρχουν διαφορετικές μέθοδοι εισαγωγής πληκτρολογίου για smartphone:

  • Κανονική πληκτρολόγηση

  • Πληκτρολόγηση αντίχειρα

  • Κανονική σάρωση

  • Συμπαγής πληκτρολόγηση (T9)

Η διπλή ολίσθηση εισόδου έχει εξερευνηθεί πολύ λίγο. Το μόνο πληκτρολόγιο που βρήκα για να εφαρμόσω μια καλή λύση εργασίας είναι πληκτρολόγιο 69.

Γιατί είναι τόσο ασυνήθιστο;

  • Η εφαρμογή είναι δύσκολη. Η πολλαπλή εισαγωγή σημαίνει ότι υπάρχουν διαδικασίες συγχρονισμού που γεμίζουν ένα buffer εισόδου που πρέπει να ερωτηθεί για υποψήφιους ολοκλήρωσης σε κάθε νέο συμβάν.

  • Απαιτεί κάποια μάθηση από τον χρήστη, κάτι που το καθιστά μη ελκυστικό, καθώς όσο υψηλότερη είναι η καμπύλη μάθησης τόσο μικρότερη είναι η διατήρηση του χρήστη.
