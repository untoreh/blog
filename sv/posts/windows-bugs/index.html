Varför Windows är ett berg av buggar som inte går att fixa

Windows (11) gnäller.

Här är en lista med anledningar till varför fönster aldrig kommer att bli bättre än skiftande sand.

Den är full av telemetri

Hur många slutpunkter pingar OS? hur ofta? Kan telemetri inaktiveras helt? Antagligen inte. OS är inte utvecklat med telemetri inaktiverad, det kommer att drabbas av några konstiga timeouts/stoppbeteenden eftersom det förväntar sig att MS-slutpunkterna är nås.

Kryptiska felkoder med ospecificerade orsaker

Allt Windows ger dig när något går snett är en felkod, nästan aldrig ett felmeddelande, och om det gör det är det obegripligt eftersom det refererar till någon körtid som med största sannolikhet är stängd källa.

ACL överallt

Naturligtvis är ACL den bättre säkerhetsmodellen. Lycka till med att ta reda på var och vilka behörigheter som krävs när du får ett tillstånd nekat fel.

Förare

Windows har inte känt igen den här enheten...Så hur fixar du detta? Kan jag peta någonstans snyggt snälla? Jag nämner inte ens förarens signaturer.

Användarstatus

Användardata lagras i%APPDATA%local ochroaming . Om jag bläddrar i de två mapparna ser jag inte tydliga definitioner, vissa appar lagrar cache i en mapp trots att det är för konfiguration andra gör tvärtom...Har windows ens ordentliga konventioner?

Windows har ingen PATH

Varje binär är vanligtvis lagrad i sin egen katalog (tillsammans med alla leverantörsberoenden). Om du försöker generalisera sökvägen skulle du sluta med en lista med tusentals kataloger där binärer lagras, åtminstone finns detscoop.

Windows-uppdateringar pressas...mycket argt

Tydligen kan det hända att Windows skjuter dåliga uppdateringar, som än drar tillbaka, eller uppdateringar som inte ens är för plattformen som de skjuts till! Det nämns inte heller att Windows-uppgraderingar är atomära, så varje gång du utför en uppgradering kastar du tärningen.

UI

Fönsterskalet är mer som enkiosk än ett UI kan du trycka på knappar och klicka på saker, så länge det är sanktionerat av Windows UI-teamet finns det inget riktigt API för genvägar. Varje applikation för att konfigurera kortkommandon är ett hack.
