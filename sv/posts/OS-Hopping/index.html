OS Hopping

En kort sammanfattning av operativa system som jag har använt

När jag fick min första persondator kom den med Windows XP (13 år). Efter några krångel fick jag lära mig att installera om windows ... Det var inte lika plug and play som idag ... Min första erfarenhet av linux var med PCLinuxOS som tydligen är går fortfarande eftersom den erbjöd en fullständig grafisk installation. Efter det gick jag direkt in Arch Linux . Jag spelade med Arch Linux ett tag (som att få gnome -gränssnittet att se ut som MacOS eller Windows Vista genom gnome look mods ), gick sedan tillbaka till Windows eftersom spel .. Från Windows hanterade jag några personliga linux -servrar ett tag med centos/debian. Efter några år gick tillbaka till Linux, först försökte jag Alpint , men dess förråd var inte (ännu?) tillfredsställande för en skrivbordsupplevelse (pulseaudio var fortfarande i testning repo).

Sen försökte jag Ogiltig Linux ett tag, men jag använde musl baserad version som inte hade bra supportpaket klokt, så jag körde ogiltiga linux plus appar i behållare på en i3wm baserad miljö. Men containeriserade grafiska applikationer var ännu inte utbredda, lösningar som flatpak snapd eller firejail var ännu inte utbredda eller till och med stabila, så jag bytte till Ubuntu och installerade en ZFS RAID-Z ( lever farligt ) rotinstallation. Som skrivbordsmiljöer behöll jag fortfarande i3wm och gick sedan till svaj orsaka wayland, och sedan KDE -plasma eftersom jag inte orkade längre med kullen systemd -enhetsfiler som måste vara användare lyckades bara få den minimala fungerande förväntade installationen med någon av kakelplattformarna.

KDE är bra, och jag anser att det är den blödande kanten av skrivbordsmiljöer, men det är också stort och med många appar som saknar bra underhåll, så mitt råd är att inte försöka hålla mig till en K* only GUI -miljö och använd bara det som fungerar, KDE har bra stöd för GTK -styling. KWin har mycket bra anpassningsalternativ och du kan binda snabbtangenter för att få det att fungera väldigt nära a flyta-först kakelplattor WM. Om du vill ha en kakel-först för WM -plattan finns det ett par tillägg, som dock inte fungerar riktigt bra på wayland (i skrivande stund).

På sistone har jag bytt tillbaka till Windows (10) .. på nytt , inte i första hand eftersom spel eftersom jag ännu måste hitta tid och motivation att spela igen, men lite av nyfikenhet för en WSL baserad Linux -miljö i kombination med bra integration för grafiska appar ( WSLg). I skrivande stund är tyvärr Windows 10-upplevelsen en de-facto nedgradering från Linux KDE. Windows med dess mus centrerad UX verkar inte ha något som är inriktat på anpassning av djupa kortkommandon. Det är oklart om av Windows -kultur genvägar rynkar pannan, sett som ett säkerhetshål eller så är det bara en allvarlig begränsning av fönsterskalet, som tvingar anpassningar genom registeröverträdelser. Powertoys har fortfarande en lång väg att gå för att tillgodose alla behov hos en person som vill anpassa sin miljö till sina vanor (inte säker på att jag kan betraktas som en kraftanvändare ..). Ändå kan jag vara ganska flexibel och med ett par tredjepartsprogram har miljön blivit något acceptabel.
