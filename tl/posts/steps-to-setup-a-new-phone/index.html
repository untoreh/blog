Mga hakbang upang mag-set up ng isang bagong telepono

Sundin ang mga hakbang upang ma-de-bloat ang isang stock android phone

  • Hanapin ang telepono batay sa iyong mga kagustuhan, mga deal sa bodega ng amazon o pag-aayos ng aliexpress na may pinakamahusay na putok para sa buck AFAIK.

  • Maaaring mangailangan ang telepono ng isang unlock code mula sa gumawa

  • Mag-check in sa fastbootoem unlock kung nangangailangan ito ng isang code

  • Isaalang-alang kung ang isang OEM stock firmware ay dapat na mai-flash mula sa fastboot bago i-flashing ang isang pasadyang ROM (suriin ang flashfile.xml)

  • Mag-flash ng isang pagbawifastboot flash recovery twrp.bin

  • Mag-boot sa recovery flash rom

  • Flash microG sa pamamagitan ng hindi opisyal na script ng pag-setup

  • Flash magisk

  • Uninstaller ng magisk module upang alisin ang mga module na nagdudulot ng mga bootloop mula sa paggaling

  • Siguro mag-flash ng isang pasadyang kernel

  • Suriin ang pagsubok sa sarili ng microG upang makita kung ano ang kailangang mai-configure (apk spoofing, unified nlp, atbp)

  • I-install ang xposed gamit ang magisk

  • I-configure ang lucky patcher

  • Mag-install ng mga merkado: f-droid, aurora, aptoide

  • Palitan ang mga app: kalendaryo-> etar, mga mensahe-> signal (session?), Mail-> deltachat (o wala ..), keyboard-> anysoftkb, files-> mixplorer, browser-> bromite / firefox, youtube-> newpipe, camera-> bukas na kamera, musika-> vynil / phonograph, workspaces-> kanlungan

  • Mga app ng baterya: mas mahusay na mga baterya, ihinto ang pagsingil ng baterya (hindi dapat kailanganin), mag-berde
