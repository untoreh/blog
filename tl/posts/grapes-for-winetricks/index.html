ubas para sa winetricks

Isang layer na batay sa pag-set up para sa mga recipe ng winetrick

Alak ay ginagamit upang magpatakbo ng mga programa ng windows sa linux. Gumagana ang runtime sa pamamagitan ng muling pagpapatupad dlls upang maging tugma sa linux. Gayunpaman ang software ay hindi lamang nangangailangan ng mga pangunahing aklatan, may mga dependency na kailangang mai-install para sa bawat piraso ng software.

Sinusuportahan ng alak ang iba't ibang mga bersyon ng windows ... (lahat ng mga ito?), Samakatuwid kinakailangan na gumamit ng iba't ibang mga kapaligiran upang matiyak ang pagiging tugma sa pagitan ng mga bersyon ng software at windows.

Ang ilang mga dependency ay maaari ding maging hindi tugma sa bawat isa, samakatuwid ang paggamit ng mga prefiks ng alak . Ang paggamit ng mga prefix ng alak ay nangangahulugang kadalasan ang bawat aplikasyon ay may sariling kapaligiran, (tulad ng mga lalagyan), na nangangahulugang iyon lahat ng mga dependency kailangang muling mai-install para sa bawat bagong programa . Wala ka ring, iba't ibang mga programa ay maaaring ibahagi ang parehong kapaligiran, ngunit tulad ng nakasaad, maaaring lumitaw ang mga problemang mahirap i-debug.

Awtomatiko ng Winetricks ang pag-install ng mga dependency (tulad ng .NET at DirectX na mga pakete), gayon pa man, kahit na scripted, ang mga pag-set up ay maaari pa ring magtagal.

At kahit na ang pangunahing kaso ng paggamit nito ay upang paganahin ang mga pag-upgrade ng atomic para sa mga OS na naka-install sa hubad na metal, maaari itong magamit upang mabuo mga prefiks ng alak na may paunang naka-install na mga koleksyon ng mga dependency, tulad ng flatpak na gumagamit ng parehong pamamaraan upang maibahagi ang mga runtime sa pagitan ng iba't ibang mga application, mabisa maaaring mayroong iba't ibang flatpak runtime ng alakpara sa mga pamilya ng software na nagbabahagi ng karamihan ng mga dependency; ang natitirang tanong lamang ay kung mayroong anumang mga problema sa lisensya na namamahagi ng ganoong uri ng mga runtime sa kung ano ang magiging epektibo preinstalled windows software . : S
